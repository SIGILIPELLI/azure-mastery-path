# Level 1 · Entry <span class="level-badge">Foundations</span>

Goal: get comfortable with the Azure Portal and the `az` CLI, understand
identity and access control, stand up compute/storage/networking/database
building blocks, and ship a small end-to-end web app.

## Modules

1. [Setup & Azure CLI/Portal](01-setup-cli-portal.md)
2. [Azure AD & RBAC / Security Basics](02-azure-ad-rbac-security.md)
3. [Virtual Machines](03-virtual-machines.md)
4. [Blob Storage](04-blob-storage.md)
5. [Virtual Networks Basics](05-virtual-networks.md)
6. [Azure SQL Database / Cosmos DB Basics](06-azure-sql-cosmos-db.md)
7. [Azure Functions](07-azure-functions.md)
8. [Azure Monitor & Log Analytics](08-azure-monitor-log-analytics.md)
9. [ARM Templates / Bicep Intro](09-arm-bicep-intro.md)
10. [Capstone Project](10-capstone-project.md)

By the end of this level you'll be able to create and secure Azure resources
from the CLI, stand up a VM and a storage account, wire up a virtual network,
run a managed database and a serverless function, watch your resources with
Azure Monitor, describe infrastructure as code with Bicep, and deploy —
then tear down — a complete small web application.
