# Azure Mastery Path

A free, structured, module-wise Microsoft Azure training program — entry
level to master level, with runnable `az` CLI commands and config snippets
in every module and a hands-on project at the end of each level. Focused on
Azure's free-tier services throughout, with explicit cleanup steps so
practicing never turns into a surprise bill.

**Live site:** https://sigilipelli.github.io/azure-mastery-path/

## Contents

- **Level 1 · Entry** — account/CLI setup, Entra ID (Azure AD) & RBAC, virtual machines, blob storage, virtual networks, Azure SQL/Cosmos DB, Azure Functions, Azure Monitor & Log Analytics, ARM/Bicep, and a capstone deploying a full Static Web App + Function + Cosmos DB app (with teardown)
- **Level 2 · Intermediate** — App Service, storage deep dive, networking deep dive, AKS basics, Azure DevOps CI/CD, managed identities & Key Vault, containers, Cosmos DB deep dive, advanced Functions
- **Level 3 · Advanced** — advanced networking, IaC at scale, advanced AKS, event-driven architecture, API Management, governance, high availability/DR, GitOps, cost optimization
- **Level 4 · Master** — enterprise landing zones, governance at scale, service mesh, hybrid cloud (Azure Arc), advanced observability, performance engineering, security (Sentinel/Zero Trust), data platform at scale, FinOps
